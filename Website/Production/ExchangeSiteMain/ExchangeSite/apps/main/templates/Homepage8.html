Welcome

Welcome to the Bitsomething exchange homepage! The site is currently under construction, and this sample news piece will serve as an introduction. I'm Mal, the only current developer. This project was first started in January of 2014. It began small, when I sketched out the exchange engine code in a few weeks. I then built it up to what it is now: a nearly operational Bitcoin exchange! Most of the very basic functions are complete. Here's a list of mostly finished features:

  • User Control - Registering/Logging in
  • Basic Orders - Placing, transacting, and seeing open orders in Account
  • Guest Pages - Non-registered users can view the guest homepage and the About pages
These are the things that are currently on the to-do list:

  • Make order book more informative
  • Add Personal Control Panel
  • Add admin page options for adding orders, users, and news blocks
  • Add petition system

A Financial Transformation

Bitcoin was the first cryptocurrency; that is, a digitally stored currency. It's stored on computers, like any other piece of data, and can be transferred anywhere in the world for less fees than a credit card. It's based on a protocol called the "Blockchain," which is a registry of all transactions ever conducted. This enables Bitcoin, and its other cryptocurrency children, to be a safer, faster, and more reliable way to pay and store value.
